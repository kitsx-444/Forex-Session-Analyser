#  Forex Session Performance Analyser

## What is this program about:
This is a forex session performance analyser. It uses pandas and summarises CSV data into useful metrics traders use to analyse their performance. The metrics are, your average RR by winners, best trade,
total wins, and total losses.

## What was new here:
Used try/except to handle missing file errors gracefully
